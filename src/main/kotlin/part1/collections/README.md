## 정리
- Kotlin의 리스트는 Java의 리스트를 확장하고 읽기 전용(이뮤터블)을 통해 컴파일 시간의 안정성을 높혔다.
- 함수형, 동시성 코드, 비동기 프로그램을 작성할 때는 이뮤터블 리스트를 사용하자.
- 작은 크기의 콜렉션 : Pair, Triple
- 고정된 크기, 큰 크기 : Array 클래스
- 동적 크기 : List, Set
